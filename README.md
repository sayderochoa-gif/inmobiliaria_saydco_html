# CostaViva | Inmobiliaria de Lujo 

CostaViva es una plataforma web estática diseñada para una firma inmobiliaria boutique especializada en propiedades de lujo en el Caribe. El sitio web proyecta exclusividad, elegancia y confianza, utilizando un diseño minimalista y sofisticado.

## Vista Previa del Proyecto

La web incluye las siguientes secciones clave:

    Inicio (Hero): Impacto visual con imágenes de alta calidad.

    Nosotros: Sección detallada con Misión y Visión corporativa.

    Catálogo de Propiedades: Cuadrícula (Grid) interactiva con 9 propiedades destacadas.

    Contacto y Ubicación: Formulario funcional integrado con un mapa de Google Maps.

    Diseño Responsivo: Adaptado para dispositivos móviles y escritorio.

## Tecnologías Utilizadas

Para este proyecto se utilizaron estándares modernos de desarrollo web:

    HTML5: Estructura semántica de la página.

    CSS3: Diseño personalizado, animaciones de hover y sistema de cuadrícula (Grid Layout).

    Google Fonts: * Playfair Display: Para títulos elegantes y clásicos.

        Poppins: Para textos de cuerpo modernos y legibles.

    Imágenes de Alta Resolución: Uso de bancos de imágenes profesionales (Unsplash/Pexels).


### Paleta de Colores

    Negro Azabache (#1a1a1a): Para textos y elegancia técnica.

    Dorado Riviera (#c5a059): Detalles de lujo, botones y acentos.

    Blanco Humo (#f4f4f4): Fondos limpios para dar "aire" al diseño.

### Instalación y Uso

    Clona o descarga este repositorio.

    Asegúrate de que los archivos index.html y style.css estén en la misma carpeta.

    Abre el archivo index.html en cualquier navegador web moderno (Chrome, Edge, Safari, Firefox).
